# Construccion-de-aplicacion-de-software
Proyecto backend desarrollado con Spring Boot y Java 17.

## Tecnologías
* Java 17
* Spring Boot 3
* H2 Database
* Maven

## Ejecución de Pruebas Unitarias
Este proyecto incluye pruebas unitarias implementadas con JUnit 5 y Mockito, además de análisis de cobertura con JaCoCo.

### Pasos para verificar la cobertura:
1. Abrir una terminal en la raíz del proyecto.
2. Ejecutar el comando: `mvn test`
3. Los reportes de cobertura se generarán en la carpeta `target/site/jacoco/index.html`.
